STEAM Discovery Lab.

About Page

Page delivers mission of the website and parent organization, while displaying creator's social media and projecting overall sense of community and accessibility. Users can visit other main pages from the navigation bar in the header and search for any content via the Search bar in the top right corner. Searching consists of filtering each page of the site for the specified word(s).

Tools Page

Allows users to filter through educational materials (physical and virtual), in order to learn more about makerspace tools. Functional filter capability by material type and by educational purpose (coding/robotics, electronics, design). since users can scroll through a long list of materials, I added a scroll-up button in the bottom right corner. Each material type also contains subcategories, accessible by the + signs.

Projects Page

Teachers can search for lesson plans based on Product and Subject filtering. At the top of the screen, they can filter project types by clicking on the "Subject" and "Tool" dropdown menus, in the green banner. The page will automatically display lessons adhering to chosen filters. The page also contains various lesson subject type, with each redirecting to separate pages with specific DIY lessons.
